This is a group of programs I created as a small personal project to track the time I spend working on my projects.

<h1><b>How to use:</b></h1>
Double click the .exe corrosponding with the action you wish to do.
<br/>
<ul>
  <li>To create an entry, click the NewEntry.exe</li>
  <li>To end an entry, click the FinishEntry.exe</li>
  <li>To see your time spend, click the ReadEntrys.exe</li>
</ul>
